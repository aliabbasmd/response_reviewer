# Response to Reviewers: LLM Informatics Research Pipeline

This repository hosts the complete, 39-step computational pipeline designed for the Frontiers manuscript. It is structured to provide full transparency and reproducibility for the journal's peer-review process.

## âš ï¸ Important: Sequential Execution
To fully replicate the study's findings, **the scripts must be executed in numerical order (Step 1 through Step 39).** * Each script generates specific data structures, metadata, or statistical variables that are required by the subsequent step.
* Skipping steps or running them out of order will result in file-path errors or missing variable dependencies.

For reviewers who wish to inspect the final outputs without running the entire 39-step pipeline, the **`samples/`** directory contains:
* **Partially Completed Notebooks:** Documented snapshots of the analysis at key milestones.
* **Worked-Out Results:** Final HTML reports and Jupyter Notebooks containing the completed visualizations, statistical tables, and model performance metrics as they appear in the manuscript.
* **High-Resolution Figures:** The raw visual outputs generated by the pipeline.

## ðŸ§¬ Pipeline Phase Overview
1. **Foundation (Steps 1-9):** Data generation, verification, and distribution analysis.
2. **LLM Integration (Steps 10-20):** Embedding extraction (BERT/Llama), comparative visualization, and complexity mapping.
3. **Classification Logic (Steps 21-27):** Cluster performance, diagnostic cutoffs, and AUC significance testing.
4. **Regression & Drivers (Steps 28-39):** Comparative regression analysis, MAE diagnostics, and identifying structural drivers of LLM performance.

## Data Dictionary (UCI Heart Disease Dataset)
This table describes the clinical variables utilized in the T5-assisted analysis and traditional ML pipelines.

| Variable | Type | Description | Mapping / Units |
| :--- | :--- | :--- | :--- |
| **age** | Continuous | Patient age | Years |
| **sex** | Categorical | Biological sex | 1: male, 0: female |
| **cp** | Categorical | Chest pain type | 1: typical, 2: atypical, 3: non-anginal, 4: asymptomatic |
| **trestbps** | Continuous | Resting blood pressure | mm Hg |
| **chol** | Continuous | Serum cholesterol | mg/dl |
| **fbs** | Categorical | Fasting blood sugar > 120 mg/dl | 1: true; 0: false |
| **restecg** | Categorical | Resting ECG results | 0: normal, 1: ST-T wave abnormality, 2: LV hypertrophy |
| **thalach** | Continuous | Max heart rate achieved | beats per minute |
| **exang** | Categorical | Exercise induced angina | 1: yes; 0: no |
| **oldpeak** | Continuous | ST depression | Induced by exercise relative to rest |
| **slope** | Categorical | Peak exercise ST segment slope | 1: upsloping, 2: flat, 3: downsloping |
| **ca** | Categorical | Number of major vessels | 0-3 colored by flourosopy |
| **thal** | Categorical | Thallium stress test | 3: normal, 6: fixed defect, 7: reversible defect |
| **outcome** | Label | Heart Disease Diagnosis | Presence (1-4) or Absence (0) |


## Dataset Information
The analysis files and worked-out examples for the UCI (University of California Irvine) heart disease dataset are located in the `/uci` folder. This directory includes both the interactive research notebooks (.ipynb) and static reports (.html).
